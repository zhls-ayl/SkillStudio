# 当前能力与后续 Roadmap

## 已落地能力
根据当前代码实现，项目已经具备以下核心能力：
- 多代理检测与本地 Skills 扫描
- 三栏式 Dashboard / Detail / Sidebar 原生界面
- `SKILL.md` 解析、编辑与 Markdown 渲染
- 从 Git 仓库安装 Skills，并写入规范目录与锁文件
- 注册表浏览与安装
- 自定义仓库配置、同步、浏览与本地安装
- 技能更新检查、提交哈希缓存与差异链接
- 文件系统监听与自动刷新
- 主题切换、应用版本检查与自更新入口
- 打包脚本、Release workflow 与 Homebrew 发布链路

## 明确存在但尚未落地的方向
以下方向在旧文档、代码注释或产品意图中出现过，但当前仓库并未完整交付：
- Project-level Skills 管理
- Create Skill Wizard / 模板化创建流程
- 更完整的搜索与多维过滤
- 菜单栏快速入口
- 更深入的批量操作能力
- 进一步完善的发行分发体验（如签名、公证、DMG）

这些项目在重新设计前，不应在 README 或对外文档中表述为“已支持”。

## 维护原则
- 本文件记录“当前已交付能力”和“明确未实现项”，不充当愿景海报
- 如果某项能力只是想法、没有代码落点，也不要写成正式 Roadmap
- 当能力已真正落地时，应同时更新 `README.md`、本文件，以及相关专题文档
