# Claude Code 协作说明

本文件仅作为 Claude Code 的兼容入口，实际仓库规范以 `AGENTS.md` 为准。
如果 `CLAUDE.md`、仓库文档与真实实现之间存在冲突，应以 `AGENTS.md`、代码、测试、脚本为事实来源。

## 建议阅读顺序
- 开始任何改动前，先阅读 `AGENTS.md`
- 需要项目背景时，按顺序阅读：`README.md` → `docs/Index.md` → 对应专题文档
- 需要判断真实行为时，继续核对相关实现、测试、脚本与 workflow

## 最低要求
- 不根据旧文档、历史描述或命名猜测当前实现
- 涉及目标不清、范围扩散、高风险改动时，先与用户确认再执行
- 修改代码后优先补相邻测试；若未补测试，需要明确说明原因
- 修改用户可见行为、路径、脚本、发布流程或文档入口后，同步更新相关文档
- 若发现与 `AGENTS.md` 不一致，应优先修正兼容说明，不保留两套规则
